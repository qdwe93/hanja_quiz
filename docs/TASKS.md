## 상태와 우선순위

- `[x]`: 산출물을 확인한 완료 작업
- `[ ]`: 미착수 또는 진행 중이며 완료 검증 전인 작업
- **P0**: MVP 출시를 막는 필수 작업

체크박스는 코드가 존재한다는 이유만으로 완료하지 않는다. 각 항목의 **완료 조건**과 연결된 검증을 모두 충족한 뒤 체크한다.
